# PROJECT ZOMBOID

## The Hive

### Описание структурной части системы модов для Project Zomboid

Вся структура состоит из маленьких сфер, объединенные одним смыслом, сферы могут пересекаться друг с другом, например Venichles могут пересекаться с Books

Список сфер:

- **Gastronomy** — сфера, отвечающая за пищу и напитки
- **Arsenal** — сфера, отвечающая за оружие и броню
- **Tweaks** — сфера, отвечающая за мелкие настройки и улучшения существующей игры
- **Garage** — сфера, отвечающая за средства передвижения
- **Archive** — сфера, отвечающая за литературу
- **Map** — сфера, отвечающая за карту
